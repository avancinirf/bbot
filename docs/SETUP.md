# BBOT – SETUP

Este documento explica como configurar e executar o BBOT em ambiente local, com foco em Mac (M1) e Docker.

---

## Pré-requisitos

- Git instalado.
- Docker instalado e em funcionamento.
- Node.js (apenas se desejar rodar frontend em modo dev fora do container).
- Python 3.11 (apenas se desejar rodar backend fora do container).
